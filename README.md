# brygg

Denna mjukvara är en enkel CLI-klient till [Punk API](https://punkapi.com)

## Installation

Klona detta repository och kör

`> bundle install`

## Usage

För att starta programmet

`./bin/brygg`

Följ sedan instruktionerna i terminalen, välj med piltangenterna.

## Testing

För att köra tester

`rake test`
